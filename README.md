# NXOpen-Geometry-Automation
Geometry automation utilities developed using Siemens NX Open API and C#.

## Overview

This project demonstrates the creation of basic geometry in Siemens NX using the NX Open API and C#.

## Features

- Create a new part
- Create construction points
- Create feature points
- Create associative lines
- Create associative arcs
- Save the part

## Technologies

- C#
- Siemens NX Open API
- Visual Studio 2022

## Project Structure

Program.cs
FileUtility.cs
GeometryUtility.cs

## Output

- Created_File.prt
- Created Geometries
- Save Geometries
